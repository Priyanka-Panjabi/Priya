# Priyanka Panjabi — Portfolio

Personal portfolio site for Priyanka Bhandari, Senior Frontend Engineer.

**Live site:** [priyanka-panjabi.github.io](https://priyanka-panjabi.github.io)

## Built With

- **[Astro](https://astro.build/)** - Static site generator
- **[Tailwind CSS v4](https://tailwindcss.com/)** - Utility-first CSS framework
- **TypeScript** - Type-safe configuration

## Local Development

```bash
git clone https://github.com/Priyanka-Panjabi/Priyanka-Panjabi.github.io.git
cd Priyanka-Panjabi.github.io
npm install
npm run dev
```

## Updating Content

All content lives in one file: `src/config.ts`

- Personal info, tagline, social links
- About section bullets
- Skills grouped by category
- Work experience with impact statements
- Education
- Projects (add here to unlock the Projects section)

## Deployment

Pushes to `main` auto-deploy via GitHub Actions to GitHub Pages.
